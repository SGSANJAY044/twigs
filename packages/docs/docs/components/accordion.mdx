### Props

#### Accordion

| Property      | Description                                 | Type                | Default |
| ------------- | ------------------------------------------- | ------------------- | ------- |
| type          | ability to open one or more accordion       | `single` `multiple` | single  |
| size          |                                             | `sm` `md`           | sm      |
| defaultValue  | uncontrolled default selected accorion item | boolean             | false   |
| value         | controlled selected accordion item          | boolean             |         |
| onValueChange | callback function for state changes         |                     |         |

#### AccordionItem

| Property | Description         | Type            | Default |
| -------- | ------------------- | --------------- | ------- |
| value    | identifier for item | single,multiple | single  |
