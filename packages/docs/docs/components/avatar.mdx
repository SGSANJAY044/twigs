### Props

#### Avatar

| Property | Description      | Type                                             | Default |
| -------- | ---------------- | ------------------------------------------------ | ------- |
| size     | size of avatar   | `xs` `sm` `md` `lg` `xl` `2xl` `3xl` `4xl` `5xl` | sm      |
| src      | image URL        | string                                           |         |
| name     | name of the user | string                                           |         |
| rounded  | border radius    | `xs` `sm` `md` `lg` `xl` `2xl` `3xl` `full`      | full    |
